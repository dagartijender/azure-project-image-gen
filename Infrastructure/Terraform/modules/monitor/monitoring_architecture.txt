Send diagnostics from AKS, Front Door, Application Gateway, Azure Firewall, Service Bus, Storage, PostgreSQL, Key Vault, and Azure OpenAI to the same Log Analytics Workspace.

Pod restart rates and node readiness.

Front Door origin health and latency.
PostgreSQL CPU, storage consumption, and connection count.
Service Bus dead-letter queue growth.
Storage availability and transaction errors.
